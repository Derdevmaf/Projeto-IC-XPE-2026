Aqui estão 50 projetos PBL para a disciplina de Programação em Python, organizados em ordem crescente de complexidade:

---

1.  **Personal Greeter:** Crie um programa que peça o nome do usuário e o cumprimente com uma mensagem personalizada. (Conceitos: `input()`, `print()`, variáveis)
2.  **Simple Arithmetic Calculator:** Desenvolva uma calculadora que execute as quatro operações básicas (soma, subtração, multiplicação, divisão) com dois números fornecidos pelo usuário. (Conceitos: `input()`, `int()/float()`, operadores aritméticos)
3.  **Unit Converter (Celsius to Fahrenheit):** Crie um programa que converta uma temperatura de Celsius para Fahrenheit. (Conceitos: Variáveis, fórmulas, `print()` formatado)
4.  **Age in Dog Years:** Peça a idade de uma pessoa e calcule sua idade equivalente em "anos de cachorro" (1 ano humano = 7 anos de cachorro). (Conceitos: Aritmética, formatação de saída)
5.  **Conditional Fortune Teller:** Um programa que peça um número ao usuário e, com base nele (e.g., par/ímpar, maior/menor que 10), exiba uma "fortuna" diferente. (Conceitos: `if/elif/else`, operadores de comparação)
6.  **Basic Tip Calculator:** Calcule a gorjeta e o valor total da conta, dado o valor da conta e uma porcentagem de gorjeta. Adicione opções para diferentes porcentagens de gorjeta. (Conceitos: Cálculos, `if/else`, porcentagens)
7.  **Number Guessing Game (Player Guesses):** O programa "escolhe" um número aleatório (entre 1 e 100) e o usuário tenta adivinhar, recebendo dicas "maior" ou "menor". (Conceitos: `random`, `while` loop, `if/else`)
8.  **Password Generator (Simple):** Gere uma senha aleatória de um comprimento específico, usando apenas letras minúsculas e números. (Conceitos: `random`, `string` module, loops)
9.  **Countdown Timer:** Implemente um temporizador que conte de um número fornecido pelo usuário até zero, exibindo cada segundo. (Conceitos: `time` module, `for` loop, `print` com `end="\r"`)
10. **Shopping List Manager (Text File):** Permita ao usuário adicionar itens a uma lista de compras e salvá-la em um arquivo de texto simples. Ao iniciar, o programa deve carregar a lista existente. (Conceitos: Listas, file I/O - `write()/read()`)
11. **To-Do List Manager (Memory Based):** Crie uma aplicação de linha de comando para gerenciar uma lista de tarefas: adicionar, visualizar, marcar como concluída e remover tarefas. (Conceitos: Funções, listas, `while` loop, menus de interação)
12. **Rock-Paper-Scissors Game:** Um jogo de Pedra-Papel-Tesoura contra o computador. (Conceitos: Funções, `random.choice()`, lógica de jogo com `if/elif/else`)
13. **Word Palindrome Checker:** Receba uma palavra ou frase e verifique se é um palíndromo (lê-se da mesma forma de trás para frente). (Conceitos: String slicing, funções, métodos de string)
14. **Prime Number Checker:** Crie uma função que verifica se um número fornecido pelo usuário é primo. (Conceitos: Funções, loops, operadores de módulo)
15. **Multiplication Table Generator:** Peça um número ao usuário e exiba sua tabuada de multiplicação de 1 a 10. (Conceitos: `for` loop, formatação de string)
16. **Simple Quiz Game:** Desenvolva um jogo de perguntas e respostas de múltipla escolha. Armazene perguntas e respostas em um dicionário. (Conceitos: Dicionários, loops, contagem de pontuação)
17. **Text-Based Adventure Game (Basic):** Crie um jogo de aventura simples onde o jogador escolhe caminhos e as decisões afetam a história. (Conceitos: Funções, dicionários para estados/salas, `if/else` encadeados)
18. **Contact Book (Memory Based):** Um programa para adicionar, visualizar e buscar contatos (nome, telefone, email) armazenados em um dicionário. (Conceitos: Dicionários aninhados, funções, manipulação de dados)
19. **Expense Tracker (Text File Persistence):** Registre despesas diárias com data e descrição. Salve as despesas em um arquivo de texto e permita visualizar o total de gastos. (Conceitos: File I/O (append), listas de dicionários, `datetime` module)
20. **Hangman Game (Text-Based):** Implemente o jogo da forca, onde o programa escolhe uma palavra e o usuário tenta adivinhar letra por letra. (Conceitos: Listas, strings, loops, `if/else`, contagem de tentativas)
21. **Student Grade Management (CSV File):** Leia e escreva dados de alunos (nome, notas em disciplinas) de um arquivo CSV. Calcule a média de cada aluno. (Conceitos: `csv` module, listas de dicionários, processamento de dados)
22. **Library Management System (Basic OOP):** Crie classes `Livro` e `Membro` com atributos e métodos básicos (e.g., `emprestar_livro`, `devolver_livro`). (Conceitos: OOP - classes, objetos, atributos, métodos)
23. **Inventory Management System (Basic OOP):** Desenvolva classes para `Produto` e `Estoque`, permitindo adicionar, remover e listar produtos. (Conceitos: OOP, listas de objetos, interação entre classes)
24. **Tic-Tac-Toe Game (Text-Based, Improved Logic):** Implemente o jogo da velha com lógica para verificar vencedor e empates, usando uma representação da grade. (Conceitos: Listas aninhadas, funções para verificação de vitória, turnos)
25. **Automated File Sorter (by Extension):** Um script que organiza arquivos em um diretório, movendo-os para subdiretórios com base em suas extensões (e.g., ".txt" para "Textos", ".jpg" para "Imagens"). (Conceitos: `os` module, loops, manipulação de arquivos e diretórios)
26. **Simple GUI Calculator (Tkinter):** Construa uma calculadora básica com interface gráfica usando Tkinter. (Conceitos: Tkinter - widgets, layout, event binding)
27. **Digital Clock with GUI (Tkinter):** Crie um relógio digital em tempo real com uma interface gráfica simples usando Tkinter. (Conceitos: Tkinter, `datetime`, `time` module, atualização dinâmica de widgets)
28. **Password Manager (Basic Encryption/Decryption):** Um programa que armazena senhas em um arquivo, usando uma biblioteca simples de criptografia/descriptografia (e.g., `cryptography.fernet`) para proteção básica. (Conceitos: File I/O, módulos de segurança, OOP para senhas)
29. **URL Shortener (Local Dictionary based):** Crie um programa que "encurte" URLs, gerando um código único para cada URL e armazenando o mapeamento em um dicionário (ou arquivo JSON). (Conceitos: Dicionários, `random`, validação de input)
30. **Basic Web Scraper (Static Page):** Escreva um script para extrair dados específicos (e.g., títulos de artigos) de uma página web estática usando `requests` e `BeautifulSoup`. (Conceitos: `requests`, `BeautifulSoup`, HTML parsing)
31. **Currency Converter (Basic API Integration):** Crie um conversor de moedas que obtenha taxas de câmbio de uma API pública gratuita (ex: exchangerate.host) e converta valores. (Conceitos: `requests` para APIs, JSON parsing)
32. **Weather App (API Integration, CLI):** Desenvolva um aplicativo de linha de comando que busca e exibe informações meteorológicas para uma cidade específica usando uma API de clima (e.g., OpenWeatherMap). (Conceitos: `requests`, JSON parsing, `argparse` para CLI)
33. **JSON Data Viewer/Editor:** Um programa para carregar, exibir e permitir a edição simples de dados em um arquivo JSON. (Conceitos: `json` module, manipulação de dicionários/listas aninhadas)
34. **Log Analyzer (Text Files):** Analise arquivos de log (texto) para encontrar padrões, contar ocorrências de erros ou extrair informações específicas usando expressões regulares. (Conceitos: `re` module, file I/O, análise de texto)
35. **Simple Drawing Application (Tkinter Canvas):** Crie um aplicativo de desenho básico onde o usuário pode desenhar linhas, retângulos ou círculos no canvas do Tkinter. (Conceitos: Tkinter Canvas, eventos de mouse, coordenadas)
36. **Blog Management System (SQLite Database):** Desenvolva um sistema CLI para gerenciar posts de blog (criar, ler, atualizar, deletar) armazenando-os em um banco de dados SQLite. (Conceitos: `sqlite3` module, SQL - CREATE, INSERT, SELECT, UPDATE, DELETE)
37. **GUI To-Do List with Persistence (Tkinter + SQLite):** Combine a interface gráfica do Tkinter com a persistência de dados do SQLite para uma lista de tarefas mais robusta. (Conceitos: Integração GUI-DB, CRUD com Tkinter)
38. **Stock Portfolio Tracker (API & GUI):** Um aplicativo GUI que permite ao usuário adicionar ações ao seu portfólio e, usando uma API de mercado de ações (e.g., Alpha Vantage), exibir seus preços em tempo real e o valor total do portfólio. (Conceitos: APIs complexas, Tkinter, persistência de dados)
39. **Image Watermarker (Pillow Library):** Crie um programa que adicione uma marca d'água de texto ou imagem a outras imagens usando a biblioteca Pillow (PIL fork). (Conceitos: `Pillow` library, manipulação de imagens, fontes)
40. **PDF Text Extractor:** Desenvolva um script que extraia todo o texto de um arquivo PDF e o salve em um arquivo de texto simples. (Conceitos: `PyPDF2` (ou similar), file I/O com PDFs)
41. **Simple Network Chat (Client-Server):** Implemente um aplicativo de chat básico usando sockets, onde um cliente e um servidor podem trocar mensagens em uma rede local. (Conceitos: `socket` module, programação de rede, `threading` para múltiplos clientes)
42. **Automated Email Sender (SMTP):** Um script que pode enviar e-mails personalizados (com ou sem anexos) para uma lista de destinatários usando o protocolo SMTP. (Conceitos: `smtplib`, `email` modules, autenticação)
43. **Data Analysis & Visualization (Pandas & Matplotlib):** Analise um conjunto de dados (e.g., um CSV com vendas, dados de clientes) usando Pandas e crie gráficos básicos (barras, linhas, dispersão) com Matplotlib. (Conceitos: `pandas` para manipulação de dados, `matplotlib` para visualização)
44. **Web Application with Flask (Basic CRUD):** Desenvolva uma pequena aplicação web com Flask para gerenciar um recurso simples (e.g., uma lista de livros ou filmes), com operações CRUD. (Conceitos: Flask - rotas, templates Jinja2, formulários HTML)
45. **Basic Machine Learning Classifier (Scikit-learn):** Carregue um dataset pequeno (e.g., Iris ou Titanic), treine um modelo de classificação simples (e.g., k-NN, Decision Tree) com `scikit-learn` e avalie seu desempenho. (Conceitos: `scikit-learn`, pré-processamento de dados, modelos de ML)
46. **Game Development (Pygame - Snake Game):** Crie o clássico jogo da "cobrinha" usando a biblioteca Pygame, com movimentação, comida e detecção de colisões. (Conceitos: `pygame` - loop de jogo, eventos, sprites, gráficos)
47. **Voice Assistant (Basic Commands):** Um assistente de voz rudimentar que pode ouvir comandos (e.g., "hora", "data") usando `speech_recognition` e responder com `pyttsx3` (text-to-speech). (Conceitos: Processamento de áudio, APIs de voz, lógica condicional avançada)
48. **Automated Data Entry (Selenium):** Crie um script que automatize a entrada de dados em um formulário web usando a biblioteca Selenium. (Conceitos: `selenium`, automação web, interação com elementos HTML)
49. **QR Code Generator/Reader:** Desenvolva um aplicativo para gerar códigos QR a partir de texto ou URLs, e também para ler informações de um código QR a partir de uma imagem. (Conceitos: `qrcode` library, `pyzbar` para leitura, manipulação de imagens)
50. **Advanced Report Generator (Multi-Source, PDF):** Construa um sistema que coleta dados de múltiplas fontes (e.g., API, banco de dados SQLite, arquivo CSV), os consolida e gera um relatório profissional em PDF usando uma biblioteca como `reportlab`. (Conceitos: Integração de múltiplas tecnologias, geração de documentos, formatação avançada)
